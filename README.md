# 使用说明

推荐用法：

使用git命令行进行操作

1. 在本地安装安装git-lfs。随便找个地方打开git bash，输入

   ```
   git lfs install
   ```

   ​

2. 将本仓库clone到本地。在希望保存本仓库的地方，输入

   ```
   git clone https://github.com/NJU-SE-15-share-review/professional-class.git
   ```

其他用法1：

1. 在github中打开所需要的文件，然后直接点击“Download”。
